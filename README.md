# StudentOS

StudentOS is a modern productivity dashboard built for students to manage their academic life in one place.
It combines task management, habit tracking, journaling, mood tracking, and focused study sessions into a clean and intuitive interface.

The goal of StudentOS is to help students stay organized, productive, and mindful throughout their daily study routine.

---

## Features

• Task Manager – create, update, and track daily tasks
• Habit Tracker – build and maintain productive habits
• Mood Tracker – log daily moods and reflect on mental state
• Pomodoro Timer – structured study sessions for focused work
• Journal System – write and revisit personal reflections
• Productivity Analytics – visualize productivity trends
• Upcoming Events – keep track of important deadlines
• AI Study Assistant – get AI-powered study help and suggestions

---

## Tech Stack

This project is built using modern web technologies:

* React
* Vite
* TypeScript
* Tailwind CSS
* Supabase

---

## Installation

Clone the repository:

git clone https://github.com/gwah1354/student_dashboard

Navigate into the project folder:

cd student_dashboard

Install dependencies:

npm install

Start the development server:

npm run dev
